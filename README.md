# GAMERS LOCATIONS

GAMERS_LOCATIONS is a list of places where the GAMERS (Guardian Angel Medical Emergency Response System) will be applicable.

Locations are classified basing on
> ID, 
> County, 
> Sub County, 
> Parish Name, 
> Village, 
> Health Facilities 

## Object format of the locations
Each location at village level is mapped to a healthcare facility.
```
{
	id: "1.0",
	county: "MATHENIKO",
	subCounty: "TAPAC",
	parishName: "KATIKEKILE",
	village: "NAKOMOL WARET ",
	healthFacilities: "Kosiroi HF",
},
  ```
  
For more information vist [gamers.doctorsarch.org](https://gamers.doctorsarch.org)
